# FE2CM Audio Player

A custom desktop wrapper for [fe2.io](https://fe2.io) built with Electron. This app allows you to control the FE2 audio streamer from your desktop with a custom UI, system tray support, and global media controls.

[![fyMN1Wu.png](https://iili.io/fyMN1Wu.png)](https://freeimage.host/)

## Features
- **System Tray:** Minimize to tray and control volume/settings from the taskbar.
- **Auto-Save:** Remembers your username and preferences.
- **Auto-Connect:** Automatically connects to the last used user on startup.

## Installation for Users
1. Go to the **Releases** tab on the right.
2. Download `FE2CM-Audio-Player.zip`.
3. Extract and run `FE2CM-Audio-Player.exe`.
